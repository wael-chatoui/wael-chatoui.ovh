# Wael Chatoui Portfolio

A modern portfolio website built with Next.js, featuring a clean design and smooth animations.

## Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Backend:** Supabase
- **Email:** Resend
- **Icons:** Phosphor Icons, Iconify

## Getting Started

### Prerequisites

- Node.js 20+ or Bun

### Installation

```bash
# Install dependencies
npm install
# or
pnpm install
# or
bun install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

### Build

```bash
npm run build
npm run start
```
